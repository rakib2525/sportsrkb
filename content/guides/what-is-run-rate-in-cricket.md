## Run rate in match context

Run rate gives a quick snapshot of scoring pace. A team scoring at 8 runs per over is moving faster than a team scoring at 5 runs per over, but the better number depends on the match format and situation.

## Partial overs

Cricket notation is not normal decimal notation. An innings at 8.4 overs has used 8 overs and 4 balls, which equals 52 balls. Treating 8.4 as a pure decimal can make the answer inaccurate.

## Related calculations

Required run rate uses the same idea during a chase. Net run rate compares scoring rate with conceded rate across a match or tournament context.

## Run rate by format

In T20 cricket, run rate pressure appears quickly because the innings has only 120 legal balls. A few quiet overs can make a chase much harder. In ODI cricket, the innings is longer, so teams may build through phases and accelerate later.

The formula stays the same in every format, but the interpretation changes. A rate of 6.00 may be strong in a difficult one-day match and too slow in a high-scoring T20 chase. Always read the number with target, pitch, wickets, and balls remaining.

## When to use a calculator

Use a run rate calculator when overs include partial notation such as 12.4. The calculator converts cricket overs safely, which prevents the common mistake of treating 12.4 as a normal decimal.
